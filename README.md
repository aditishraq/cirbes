# CirBES Website

A clean, professional website for CirBES - Circular Business and Engineering Systems AB.

## File Structure

```
/
├── index.html              # Home page
├── about.html              # About CirBES page
├── services.html           # Services page
├── projects.html           # Projects page
├── contact.html            # Contact page
├── circular-vision.html    # Circular vision page
├── assets/
│   ├── css/
│   │   └── style.css      # Main stylesheet
│   ├── js/
│   │   └── main.js        # JavaScript functionality
│   └── images/            # All images and PDFs
└── README.md              # This file
```

## Features

- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Clean Layout**: Professional, easy-to-navigate interface
- **Brand Colors**: Uses CirBES blue (#376092) and light blue (#95b3d7)
- **Smooth Animations**: Scroll animations and hover effects
- **Interactive Elements**: Tab system on About page, form validation

## Pages

1. **Home** - Overview of CirBES, projects, references, services, and focus areas
2. **About** - Company story, founder team bios, and skills
3. **Services** - Detailed service offerings including workshops and analysis
4. **Projects** - Three main projects with descriptions
5. **Contact** - Contact information and form
6. **Circular Vision** - Explanation of circular business principles

## How to Use

1. Open `index.html` in a web browser to view the website
2. All pages are linked through the navigation menu
3. Images are referenced from the `assets/images/` folder
4. PDFs for additional resources are also in the images folder

## Customization

- **Colors**: Edit CSS variables in `assets/css/style.css` (lines 8-15)
- **Content**: Edit HTML files directly to update text
- **Images**: Replace files in `assets/images/` keeping the same filenames
- **Fonts**: Change font-family in the body selector in CSS
